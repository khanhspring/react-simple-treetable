### React simple tree table

#### Advantages
- Dynamic columns
- Tree table view
- Tree line through on the rows
- Toggle all
- Light weight (only reactjs, css)
- Easy to custom

#### Usage

1. Clone the project
```
$ git clone https://github.com/khanhspring/react-simple-treetable.git
```

2. Install dependences
```
$ cd react-simple-treetable
$ npm install
```

3. Start
```
$ npm start
```
